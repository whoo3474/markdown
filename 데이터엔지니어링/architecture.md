# ETL (Extract, Transform, Load)
추츨하고 변환하고 적재하고

# DL (Date Lake)
비정형 데이터의 모음

# DW (Data Warehouse)
정형 데이터의 모음
의사 결정 및 분석에 도움을 주기 위해 분석 가능한 형태 정보 창고

# DM (Data Mart)

각각의 분야/ 서비스에서 필요한 만큼 유저/각 팀가 사용하도록 나누어 놓은 데이터
